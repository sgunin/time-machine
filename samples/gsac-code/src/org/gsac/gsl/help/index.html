GSAC enables you to search a data repository for information (metadata) about sites and instruments, and to download data files and product files from the instruments.

To learn how to use GSAC web services, including the web forms, and the GSAC API for programmatic (computer) search requests, read the UNAVCO GSAC User Guide.

To learn about what GSAC is, and to install GSAC to provide web services for a geoscience data repository, see the UNAVCO GSAC web site.
