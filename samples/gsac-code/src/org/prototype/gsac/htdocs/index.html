Data Repository GSAC Web Services

This Data Repository uses UNAVCO's GSAC software for data searches and file downloads. To begin searches for site information, click on Search Sites above, or click on Search Files to find and download instrument data files from this repository. For more details about how to use GSAC to find and download GNSS data, see the GSAC User Guide.

GSAC software provides general-purpose web services, accessing data repositories with Earth science data. GSAC is based on the idea of sites or stations with instruments making automated measurements. Examples are permanent GNSS receivers and tide gages. Instruments have associated data files which can contain direct observations (measurements), or product files derived from observations. GSAC lets you find information about sites and instruments, and find how to download data files. One GSAC can serve one or more independent data file repositories. GSAC is not highly tailored for one kind of instrument or data file type. To install GSAC see the UNAVCO GSAC web site.
